Color

There are a wide range of CSS options available to customize the colors on an html page. These colors can be applied to text, backgrounds, links, buttons, and much more. Colors can be further customized using hex, rgb(), and hsla() values.

Property CSS Description
color color: rgb(255, 0, 11); Used to set the color of foreground elements; typically used along with a color value to set the color of text.
background-color background-color: hsl(160, 50%, 10%); Used to set the color of an element's background.
border-color border-color: #fff; Long-hand border property. Sets the color of an element's border.
background-image background-image: linear-gradient(red, yellow); Specifies a background image to be used as an element's background. Can take an image url or a linear gradient function.

Text

The style of font elements can be modified with CSS too! These properties can add specific styling attributes to text such as underlinine or bold. They can also be used to import custom fonts from around the web and APIs like Google Fonts.

Property CSS Description
font-family font-family: 'Arial'; Used to set the text's primary and backup font family if not available.
font-style font-style: italic; Sets the text attribute as bold or italicized.
font-weight font-weight: 400; Specifies the weight or degree of thickness of a font.
text-transform text-transform: uppercase; Applies transformations, such as uppercase, to text.
text-decoration text-decoration: underline; Modifies the appearrance of text. Can be used to add underline, dotted or wavy decorations, etc.
font-size font-size: 16px; Used to set the size of font in pixels (px) or em.
@font-face @font-face {src: url() format();} CSS rule used to specify the location of an installed font.
letter-spacing letter-spacing: 10px; Determines the spacing between characters.
word-spacing word-spacing: 5%; Sets additional spacing (whitespace) between words.
line-height line-height: 2em; Determines the vertical area of a text box.

Size and Position

Page elements can be ordered and resized with a variety of properties specifically designed to give developers more control over the organization of a web page. Sizing properties include code that adjust the margins between objects as well as their width and height.

Property CSS Description
width width: 50%; Sets the default width in pixels or percent of an element's content-box relative to its parent.
height height: 718px; Set's the default height in pixels or percent of an element's content-box relative to its parent.
max/min-width max-width: 200px;
min-width: 150px; 		Sets the maximum or minimum width of an element's content box. Useful when the viewports can be resized.
max/min-height max-height: 315px;
min-height: 275px; 		Sets the maximum or minimum height of an element's content box.
margin margin: 0 auto; Sets the margins (the space outside of the border) for an element.
padding padding: 20px; Sets the padding (the space inside of the border) for an element.
display display: inline-block; Determines the type of box to be used for an element.
position position: fixed; Along with the left, right, top, and bottom properties, determines how an element is positioned within the flow of a document.

Responsive Layouts

Grids, flexboxes, and media elements allow you to accurately and reliably lay out webpages on a variety of screen sizes.

Property CSS Description
display display: grid;
display: inline-grid; 		Sets the display property of the element to a grid container. Required to define additional CSS grid properties.
grid-template grid-template: 50px 40px / 100px 100px; Shorthand property for grid-template-rows and grid-template-columns. Sets the number and width/height of grid rows followed by columns.
grid-template-rows grid-template-rows: 60px 2fr 1fr; Sets the number and height of grid rows. The example to the left creates 3 rows: The first is 60px tall. The second is 2/3 the remaining space. The third is 1/3 the remaining space.
grid-template-columns grid-template-columns: repeat(4, 50px); Sets the number and width of grid columns. The example to the left creates 4 identical columns, each with a width of 50px.
gap gap: 10px 15px; Sets the gutter space between grid rows and columns. Shorthand property for row-gap and column-gap.
row-gap row-gap: 20px; Sets the gutter space between grid rows.
column-gap column-gap: 15px; Sets the gutter space between grid columns.
grid-row / grid-column grid-row: 5 / span 2; Shorthand property for grid-row-start and grid-row-end or grid-column-start and grid-column-end. Specifies how many grid lines a row will span. The first value specifice the row start. The second value specifies the row end.
grid-row-start / grid-column-start grid-row-start: 1; Specifies the starting grid line for a grid row or column item. The example to the left describes a grid item that starts at row line 1 which is immediately before the first row begins.
grid-row-end / grid-column-end grid-row-end: 5; Specifies the terminal grid line for a grid row item or grid-column-item. The example to the left ends at grid row line 5. This is the line immediately before, but not inclusive of, row 5.
grid-area grid-area; 2 / 1 / span 2 / span 4; Shorthand property for grid-row and grid-column. The order is: row start, column start, row end, then finally column end.
